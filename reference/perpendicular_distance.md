# Perpendicular distance from points to a line segment

Perpendicular distance from points to a line segment

## Usage

``` r
perpendicular_distance(points, start, end)
```

## Arguments

- points:

  Nx2 numeric matrix of (x, y) points

- start:

  Numeric vector of length 2 (line start)

- end:

  Numeric vector of length 2 (line end)

## Value

Numeric vector of perpendicular distances
