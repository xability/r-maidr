# Process a single patchwork panel

Process a single patchwork panel

## Usage

``` r
process_patchwork_panel(
  leaf_plot,
  panel_name,
  panel_index,
  row,
  col,
  layout,
  gtable
)
```

## Arguments

- leaf_plot:

  The leaf ggplot object

- panel_name:

  Panel name from gtable

- panel_index:

  Panel index

- row:

  Panel row

- col:

  Panel column

- layout:

  Layout information

- gtable:

  Gtable object

## Value

Processed panel data
